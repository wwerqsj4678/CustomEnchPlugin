# GTMBPlugin

用于Minecraft（我的世界）中国版端游联机地图《联机大厅服务器模板》的配套附加包

主要功能：自定义附魔、获取隐藏物品、修改物品NBT、指令批处理、修改物品注释、防止聊天文本卡客户端、违禁词玩家检测
